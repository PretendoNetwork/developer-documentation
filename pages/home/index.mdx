# Developer Documentation

Welcome to the Pretendo Network developer documentation. Here you will find information on various libraries used to build servers, as well as information on general topics such as packet transport and game server infrastructure.

For information on the core concepts used in game servers, see the **Overview** section.

For information on the libraries used to create game servers, see the **Libraries** section.

The information found here can be seen in a less detailed Wiki-style format here https://nintendo-wiki.pretendo.network/docs/.
